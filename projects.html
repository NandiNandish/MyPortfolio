Customer Churn Prediction

A machine learning model to predict customer churn in subscription-based businesses, built from scratch using NumPy and fundamental ML principles.

The Problem

Subscription-based businesses lose significant revenue due to customer churn. Traditional churn prediction models are often black-box. This project demonstrates transparent, interpretable ML fundamentals.

The Solution

Built a custom ML pipeline from scratch using NumPy to implement classification algorithms. Integrated Pandas for data preprocessing, Matplotlib & Seaborn for visualization, and Scikit-learn for benchmarking against industry-standard implementations.

Key Features

  • Custom algorithm implementation from scratch (no black-box libraries)
  • Comprehensive data cleaning and feature engineering
  • Model evaluation with accuracy, precision, recall, and F1-score
  • Interactive Jupyter notebook with detailed explanations
  • Visualization of decision boundaries and feature importance

Impact

Achieved 87% accuracy in churn prediction. Demonstrates mastery of ML fundamentals and ability to implement algorithms from first principles without relying on abstraction layers.

Caffee Management System

A comprehensive admin interface for managing café operations, menus, inventory, and customer orders with a clean, intuitive UI.

The Problem

Café owners struggle with manual inventory tracking, menu management, and order processing. Existing solutions are expensive and complex for small businesses.

The Solution

Built a full-stack web application with an elegant admin dashboard. Features real-time inventory tracking, menu customization, order management, and sales analytics. Backend powered by Laravel with MySQL database for reliability and scalability.

Key Features

  • Responsive admin dashboard with real-time updates
  • Menu & inventory management with drag-drop UI
  • Order processing and kitchen display system
  • Sales analytics and reporting
  • Multi-user support with role-based access control
  • Mobile-responsive design for all devices

Impact

Deployed at 2 local cafés. Reduced order processing time by 40% and inventory management overhead by 60%. Customer satisfaction rating: 4.8/5.

AI Secure Lock

A modern web application demonstrating advanced React development, real-time encryption, and secure data handling with a sleek UI.

The Problem

Users need a secure, user-friendly way to manage and share sensitive information. Existing password managers are outdated and lack modern UX.

The Solution

Built a modern web app using React 18 and TypeScript for type safety. Integrated Supabase for real-time database, CryptoJS for client-side encryption, and modern UI with Tailwind CSS. Fast builds powered by Vite.

Key Features

  • Client-side encryption for maximum security
  • Real-time synchronization across devices
  • Password strength analysis and recommendations
  • Two-factor authentication (2FA) support
  • Fast, responsive UI with Vite build tool
  • User-friendly secure sharing with expiration

Technical Highlights

Demonstrates advanced React patterns, TypeScript best practices, secure authentication flow, real-time database integration, and modern frontend performance optimization techniques.
